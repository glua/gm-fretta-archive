GarryWare Notice

Thanks for downloading GarryWare! Here is a quick installation notice.

Where to install?

You need to arrange folders so that this pattern matches : garrysmod/gamemodes/garryware/info.txt

Precautions for server hosting

You should consider shifting every file from content/ folder of your gamemode to your garrysmod/ folder on your server. That way, players will download content directly to the right location in their game. My apologies for that, but the way Garry's Mod gamemodes system currently is will force you to do that, because the server will look for the files to send over as if they were physically in garrysmod/ folder.

This gamemode makes use of limited lua_run entities in certain minigames, to circumvent gamemode hooks that are not working properly. It is especially used to circumvent the use of GravGunPickup-type of hooks. Due to this, servers that have custom modules that block all lua_run type of triggers will prevent this game from working properly.

Will players get kicked out if they don't have the content?

If hosting a server with players that don't have the custom content (maps and sounds), the server will try to send them over (every custom content is provided). Also, this gamemode reguires any Source game : It only uses crucial content from shared Source files. That means any Garry's Mod owner can run GarryWare with no issues.
